درباره پروژه

سیستم مدیریت کلینیک ما با استفاده از جاوا و جاوا سوئینگ ساخته شده است تا یک برنامه دسکتاپ پاسخگو و کاربرپسند برای ارائه‌دهندگان خدمات درمانی فراهم کند. این پروژه بر ایجاد سیستمی کارآمد برای مدیریت پرونده‌های پزشکی، قرارملاقات‌ها و وظایف اداری تمرکز دارد.

فناوری‌های کلیدی:

  • جاوا SE - زبان برنامه‌نویسی اصلی
  • جاوا سوئینگ - چارچوب رابط کاربری گرافیکی برای برنامه دسکتاپ
  • JDBC - اتصال به پایگاه داده
  • MySQL - پایگاه داده سمت سرور
  • Maven - مدیریت وابستگی‌ها
